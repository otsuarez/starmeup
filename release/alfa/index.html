deploy a single package

items

  • playbook: deploy.yml
  • Makefile

makefile

Use a Makefile for abstracting the playbook execution and parameter passing.

instead of

ansible-playbook deploy.yml -e '{ "packages" : [{"name":"acme-frontend-emerging","version":"3.33.613-master-0fc53ca"}]}'

just

make deploy name=acme-web-core version=3.33.163-master-5341313

playbook

the playbook receives a json in the following format:

{
  "packages" :
  [{
    "name" : "acme-web-core", "version" : "3.33.163-master-5341313"
  }]
}

instead of

ansible-playbook deploy.yml -e '{ "packages" : [{"name":"acme-frontend-emerging","version":"3.33.613-master-0fc53ca"},{"name":"acme-web-core", "version": "3.33.163-master-5341313"}]}'

just

make deploy name=acme-web-core version=3.33.163-master-5341313
