vagrant-lxc

https://github.com/fgrehm/vagrant-lxc

The plugin is known to work better and pretty much out of the box on Ubuntu 14.04+ hosts

requerimientos

  • Vagrant 1.1+
  • lxc 0.7.5+
  • redir (if you are planning to use port forwarding)
  • A kernel != 3.5.0-17.28
dpkg -l | grep lxc
ii  lxc                                                   1.0.5-0ubuntu0.1                                    amd64        Linux Containers userspace tools

sudo apt-get install redir

osvaldo@infr-laptop8:~$ vagrant --version
Vagrant 1.6.5
osvaldo@infr-laptop8:~$

osvaldo@infr-laptop8:~$ uname -a
Linux infr-laptop8 3.13.0-24-generic #46-Ubuntu SMP Thu Apr 10 19:11:08 UTC 2014 x86_64 x86_64 x86_64 GNU/Linux
osvaldo@infr-laptop8:~$

Installation

vagrant plugin install vagrant-lxc
vagrant lxc sudoers

Quick start

vagrant init fgrehm/precise64-lxc vagrant up --provider=lxc
