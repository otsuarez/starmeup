Running commands

All commands should be executed as the rls user on the rls-jenkins server in the private cloud.

Adding a package to a repo

aptly repo add static-images <deb package>
aptly publish update  acme-static-images  s3:static-images:

Remove a package

aptly repo show --with-packages=true prod | grep acme-elvira
aptly repo remove -dry-run=true prod 'acme-elvira'
aptly repo remove prod 'acme-elvira'
aptly publish update acme-prod prod
aptly db cleanup
