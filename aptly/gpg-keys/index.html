Export gpg key

Publi—Å part of the key should be exported (gpg --export --armor) and imported into apt keyring on all machines that would be using published repositories using apt-key.

gpg --export --armor rls-team@acme.com > rls-team-pubkey.asc
gpg --export-secret-keys --armor rls-team@acme.com > rls-team-privkey.asc

rls@rls-jenkins:~$ gpg --list-keys
