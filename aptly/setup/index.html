Installation

deb repository

Native packages are available for aptly. Create the following repo file.

/etc/apt/sources.list.d/aptly.list

deb http://repo.aptly.info/ squeeze main

Update and install

sudo apt-get update
sudo apt-get install aptly

Configuration

aptly.conf

{
  "rootDir": "/var/lib/aptly",
  "downloadConcurrency": 4,
  "architectures": [],
  "dependencyFollowSuggests": false,
  "dependencyFollowRecommends": false,
  "dependencyFollowAllVariants": false,
  "dependencyFollowSource": false,
  "gpgDisableSign": false,
  "gpgDisableVerify": false,
  "downloadSourcePackages": false,
  "ppaDistributorID": "ubuntu",
  "ppaCodename": "",
  "S3PublishEndpoints": {
    "static-images": {
      "region": "us-east-1",
      "bucket": "rls-deploy-packages",
      "awsAccessKeyID": "***********",
      "awsSecretAccessKey": "***********",
      "prefix": "",
      "acl": "private"
    },
    "c2c": {
      "region": "us-west-2",
      "bucket": "c2c-rls-deploy-packages",
      "awsAccessKeyID": "***********",
      "awsSecretAccessKey": v
      "prefix": "",
      "acl": "private"
    }
  }
}

Security

The /var/lib/aptly is owned by user rls in order to avoid using sudo to perform any operation.

Upgrade

Check current version

rls@rls-jenkins:~$ aptly version
aptly version: 0.7.1
rls@rls-jenkins:~$

Check available versions.

rls@rls-jenkins:~$ sudo apt-get update
...
rls@rls-jenkins:~$ apt-cache madison aptly
     aptly |        0.8 | http://repo.aptly.info/ squeeze/main amd64 Packages
     aptly |      0.7.1 | http://repo.aptly.info/ squeeze/main amd64 Packages
     aptly |        0.7 | http://repo.aptly.info/ squeeze/main amd64 Packages
     aptly |        0.6 | http://repo.aptly.info/ squeeze/main amd64 Packages
     aptly |      0.5.1 | http://repo.aptly.info/ squeeze/main amd64 Packages
     aptly |        0.5 | http://repo.aptly.info/ squeeze/main amd64 Packages
     aptly |      0.4.1 | http://repo.aptly.info/ squeeze/main amd64 Packages
rls@rls-jenkins:~$

Install the new version.

sudo apt-get install aptly

Check version.

rls@rls-jenkins:~$ aptly version
aptly version: 0.8
rls@rls-jenkins:~$
